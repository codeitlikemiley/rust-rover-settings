## Customizing IdeaVim Keys

check list of Actions we can bind with .IdeaVim to our Keys

```sh
:actionlist
```

We can add e.g. to our `~/.ideavimrc`

```sh
nmap <leader>v :action Tool_External Tools_gvim<cr>
```
